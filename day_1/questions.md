## Day 1 Questions

1. How would you print the string `"Hello World!"` to the terminal?
```ruby
puts "Hello World!"
```
2. What character is used to indicate comments in a ruby file?
#octothorpe

3. Explain the difference between an integer and a float?
An integer is a number without a decimal point. A floating-point number has a decimal point. Floats are used when more precision is needed.

4. In the space below, create a variable `animal` that holds the string `"zebra"`
```ruby
animal = "zebra"
```

5. How would you print the string `"zebra"` using the variable that you created above?
```ruby
puts animal
```
6. What is interpolation? Use interpolation to print a sentence using the variable `animal`.
```ruby
puts "An #{animal} has stripes!"
```

7. What method is used to get input from a user?
```ruby
gets.chomp
```
8. Name and describe two common string methods:
```ruby
animal.length #returns the number of characters
animal.length.capitalize #capitalizes the first letter in the string
```
