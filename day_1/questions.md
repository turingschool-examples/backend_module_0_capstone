## Day 1 Questions

1. How would you print the string `"Hello World!"` to the terminal?

```ruby
p "Hello World!"
```

1. What character is used to indicate comments in a ruby file?

> The Octothorpe or pound symbol #

1. Explain the difference between an integer and a float?

> Interger is a whole number and a float has a fraction denoted with a decimal "."

1. In the space below, create a variable `animal` that holds the string `"zebra"`

```ruby
animal = "zebra"
```

1. How would you print the string `"zebra"` using the variable that you created above?

```ruby
p animal
```

1. What is interpolation? Use interpolation to print a sentence using the variable `animal`.

> Interpolation is the when you insert data into a string

```ruby
p "My favorite animal is the #{animal}!"
```

1. What method is used to get input from a user?

> Method `gets` gets input from a user and stores it as a string

1. Name and describe two common string methods:

> Method `.length` outputs the number of characters in a strings
> Method `.chomp` removes a line break from the strings
